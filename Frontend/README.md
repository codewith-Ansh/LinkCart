# LinkCart

LinkCart is a platform designed to provide creators and sellers with one shareable link where they can showcase multiple products, links, and profiles in a single place. It offers a clean, mobile-friendly public page for buyers, making it easy to discover and access content through one unified link. The product is built with a focus on simplicity, speed, and a creator-first experience, while remaining scalable for future features such as customization, analytics, and monetization.

## Project Setup

Navigate to the project directory
cd frontend

Install dependencies
npm install

Start the development server
npm run dev

Open the application in the browser at
http://localhost:5173

This setup is for local development only

## Project Status

The project is currently in active development. At this stage, the primary focus is on building a clean, scalable, and maintainable foundation for the LinkCart user interface.
